1. Text 文本组件

   1. 常见属性

      1. data - 必要，显示的文本内容
      2. style - 文本样式：大小、颜色、粗细、字体等
      3. textAlign - 文本对齐方式
      4. maxLines - 文本最大行数
   2. 如果想要实现富文本效果，可以用Text.rich()配合TextSpan使用，**但如果段文本以及大量重复样式文本，建议同一定义，有助于保持一致性并提升性能**。
   3. ```dart
        children: [
            Text(
                "Hello,Flutter!Hello,Flutter!Hello,Flutter!Hello,Flutter!Hello,Flutter!Hello,Flutter!Hello,Flutter!Hello,Flutter!Hello,Flutter!Hello,Flutter!Hello,Flutter!Hello,Flutter!",
                maxLines: 2,//行数限制
                overflow: TextOverflow.ellipsis,//超出最大行显示
                style: TextStyle(
                    fontSize: 50,
                    color: Colors.blue,//文本颜色
                    fontStyle: FontStyle.italic,//文本类型
                    fontWeight: FontWeight.w900,//文本粗细
                    decoration: TextDecoration.underline,//文本装饰
                    decorationColor: Colors.red,//装饰颜色
                ),
            ),
            //如果想要实现富文本效果，可以用Text.rich()配合TextSpan使用
            Text.rich(TextSpan(
                //第一个 
                text: "Hello",
                style: TextStyle(color: Colors.red,fontSize: 40,fontWeight: FontWeight.bold,decoration: TextDecoration.underline,decorationColor: Colors.amber),
                //依次接续,如果TextStyle不写样式，则会直接继承父类的Style
                children: [
                    TextSpan(text: " Flutter",style: TextStyle(color: Colors.green,fontWeight: FontWeight.normal)),
                    TextSpan(text: "!"),
                ],
            ))
        ],
      ```

2. Image 图片组件
   有多种创建方式

   1. `Image.asset()` - 加载项目目录资源，需要在配置文件中声明

   2. `Image.network()` - 加载网络资源，安卓、IOS、鸿蒙需要配置网络权限

   3. `Image.file()` - 加载本地图片

   4. `Image.memory()` - 加在内存中的图片数据

   5. ```dart
      children: [
        //需要在pubspec.yaml配置图片路径所在位置
        Image.asset(
            "assets/images/Icon-512.png",
            width: 100,
            height: 100,
            fit: BoxFit.fitWidth,
        ),
        //在安卓/IOS/鸿蒙中使用网络图片需要配置网络权限
        Image.network(
            "https://yjy-teach-oss.oss-cn-beijing.aliyuncs.com/meituan/1.jpg",
            width: 100,
            height: 100,
            fit: BoxFit.fitHeight,
        )
      ],
      ```

3. TextField 文本框组件

   1. 必须得是有状态组件中的一部分

   2. 常用属性

      1. controller - 文本框控制器，获取文本内容以及监听变化的
      2. decortation - 文本框样式
      3. style - 文字样式
      4. maxLine - 最大行数
      5. onChanged - 文本框内容发生变化后的回调函数
      6. onSubmitted - 文本框内容被提交后的回调函数

   3. ```dart
        return TextField(
            controller: controller,//编辑器控制器，获取文本内容以及监听变化的。
            obscureText: hintText.endsWith("password")?true:false,//是否隐藏内容
            // onChanged: (value) => print(value),//当文本框内容发生变化时触发
            onSubmitted: (value) => print(value),//当文本框内容提交（回车）时触发
            decoration: InputDecoration(//定制样式
                hintText: hintText,//提示文本
                contentPadding: EdgeInsets.only(left: 20),
                fillColor: const Color.fromARGB(255, 195, 194, 159),//文本框背景色
                filled: true,//是否应用文本框背景色
                border: OutlineInputBorder(
                    borderSide: BorderSide.none,
                    borderRadius: BorderRadius.circular(25),
                ),
            ),
        );
      ```
