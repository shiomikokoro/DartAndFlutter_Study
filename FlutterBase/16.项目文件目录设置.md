- lib
    - api 存放请求
    - assets 存放资源
    - components 存放公共组件
    - contants 存放常量文件
    - viewmodels 存放类型文件
    - pages 存放页面
    - routes 存放路由配置
    - stores 存放全局状态组件
    - utils 存放工具类
    - main.dart 入口
- git 上传代码
    - git init 初始化仓库
    - git add . 提交本地仓库到暂存区
        - 第二次起使用 git add -A
    - git commit -m “使用说明” 提交本地仓库
    - git remote add origin master url 添加远程仓库
        - github修改了主分支是main，根据自身情况修改
    - git push -u origin master 推送主分支到远程
        - github修改了主分支是main，根据自身情况修改
